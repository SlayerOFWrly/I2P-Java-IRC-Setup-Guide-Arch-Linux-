# I2P-Java-IRC-Setup-Guide-Arch-Linux-
An Arch Linux guide that covers a quick install of I2P (Java) and how to set up a very basic and barebones IRC server via Inspircd that tunnels into I2Ps multilayer peer 2 peer network for a more secure messaging option that even works on mobile for those looking to drop the clear web in favour of privacy and your human right to privacy.
